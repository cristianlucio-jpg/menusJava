# Proyecto: menusJava  
_Ejercicios prácticos de menús en Java_

## Descripción
Este proyecto contiene una serie de ejercicios en **Java** que implementan **menús interactivos** para resolver problemas básicos de lógica y programación.  
Forma parte de un **portafolio académico y profesional** orientado a la práctica de **estructuras de selección, modularidad y entrada/salida de datos**.

---

## Objetivos
- Comprender el uso de **estructuras de selección** (`if`, `else`, `switch`) en Java.  
- Implementar menús interactivos que permitan al usuario elegir opciones.  
- Modularizar programas dividiendo la lógica en métodos auxiliares.  
- Practicar entrada y salida de datos con `Scanner` y `System.out`.  
- Desarrollar programas organizados y bien documentados como parte de un portafolio profesional.  

---

---

## Contenido temático
- **Conceptos fundamentales** → tipos de datos, operadores, variables y constantes.  
- **Control de flujo** → estructuras condicionales (`if`, `else`, `switch`) y repetitivas.  
- **Funciones y procedimientos** → modularización mediante métodos auxiliares.  
- **Entrada y salida de datos** → interacción con el usuario usando `Scanner` y `System.out`.  
- **Aplicaciones prácticas** → menús para cálculos, registros y lógica básica.
